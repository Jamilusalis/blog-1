OPEN SOURCE

Why open source?

4-5 minutes

Open Source refers to software for which the original source code can be shared, inspected, modified and enhanced.
Open source projects embrace and celebrate principles of open exchange, collaborative participation, rapid prototyping, transparency and community-oriented development.

Open Source software is not only important to software developers but the whole globla spectrum as well. Early inventors built muh of the internet on open source technologies like Linux and Apache Web Server.

Open Source Licenses

An open source license is a license for computer software and other products that allows the source code, blueprint or design to be used, modified or under certain defined terms and conditions. These licenses may have restriction particularly regarding the expression of respect

Why Open Source?

  • Training : Most developers like open source software because it helps them become better developers. With the publicly accessible source code, they can easily study it as they learn. Developers can also share their work with others, inviting comment and critique as they develop their skills.
  • Stability : Developers prefer open source for long-term and important projects since they can be sure that their tools wont disappear or fail to disprepair if their original creators stop working on them.
  • Security : the fact that anyone can view and modify open source code, someone is likely to spot errors or omissions and correct them that original authors might have missed and because developers can work on without asking for permission from authors enbaling them to fix, update and upgrade more quickly compared to propriety software.
